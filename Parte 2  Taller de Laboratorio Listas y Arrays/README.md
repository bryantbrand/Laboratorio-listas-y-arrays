### Analsis comparativos entre listas y arrays 

**Las listas**
*Son modificables y son mejores si necesitamos hacer un proyecto y necesitamos que sea flexible *Son heterogeneas pueden tener todo tipo de variables int, string y objetos, *Son menos eficientes ya que los guarda el espacio diferentes.

**Arrays**
*No son tan versatiles son mas estaticos
*Requieren de las librerias
*Todos los elementos tienen que ser del mismo tipo de variable int/string/float.
*Son mas eficientes por su rendimiento ya que estos guardan los datos seguidos haciendo al algoritmo mas eficiente.
